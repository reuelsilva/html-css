Principais Formatações

Negrito/Destaque

Nessa frase temos um termo em destaque usando a tag semântica <strong>.

Itálico/Ênfase

Nesta frase temos um termo em itálico usando a tag semântica <em>.

Podemos criar também um texto marcado usando a tag <mark>.

Texto deletado

Podemos marcar um texto excluído com a tag <del>.

Texto sublinhado

Podemos marcar um texto com sublinhado com a tag semântica <ins>.

Texto sobrescrito

O texto sobrescrito aparece acima da linha normal, como:

x2 + y, 210

Texto subescrito

O texto subescrito aparece abaixo da linha normal, como:

H2O, O2
